⚡ Configuración inicial

Solo una vez. Necesitas un mini-servidor gratuito en Cloudflare para que las descargas funcionen directo.

1

Crear cuenta en Cloudflare

Ve a dash.cloudflare.com/sign-up y crea una cuenta gratis.

2

Crear un Worker

Ve a Workers & Pages → Create → "Hello World" → ponle nombre → Deploy.

3

Pegar el código

Toca Edit Code. Borra todo y pega el código del archivo worker.js. Toca Deploy.

4

Copia tu URL

Tu Worker tendrá una URL tipo:
https://mediagrab.tu-usuario.workers.dev
